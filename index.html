Key Project

Explainable AI Decision Support System
MSc Final Project — University of the West of Scotland, 2026

Designed and built a full-stack web application for evaluating carbon pricing policies in the energy sector. The system integrates a Flask Python backend with a React frontend via RESTful APIs, combining multi-source data pipelines, hybrid machine learning models, and interactive explainability visualisations into a single deployable product.

Data was sourced from the EU ETS, IEA, World Bank, and UK DESNZ — merged, cleaned, and feature-engineered into a unified panel dataset. Seven model families were trained with a weighted ensemble achieving R² of 0.99 on policy effectiveness scoring. SHAP and LIME layers provide stable, human-readable explanations for every prediction.
